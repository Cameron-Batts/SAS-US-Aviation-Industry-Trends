# U.S. Aviation Industry Trends using SAS Visual Analytics

## Overview
Interactive SAS Visual Analytics dashboard exploring the impact of COVID-19 on the U.S. airline industry using U.S. Department of Transportation data.

## Tools Used
- SAS Visual Analytics
- SAS Viya for Learners

## Dashboard Features
- Passenger trend analysis
- Forecasting models
- Airline employment analysis
- Profit and loss visualization
- Revenue and expense comparisons
- Interactive filtering and controls

## Key Insights
- Passenger traffic sharply declined during COVID-19
- International travel recovered slower than domestic travel
- Airlines experienced significant financial losses beginning in 2020
- Employment reductions lagged behind passenger declines
